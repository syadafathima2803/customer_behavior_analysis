Overview
This project focuses on analyzing customer behavior using Python, SQL, and Power BI. The goal is to extract meaningful insights from raw data through data cleaning, exploratory analysis, SQL querying, and interactive visualization, thereby supporting data-driven business decisions.
Dataset
The dataset contains customer demographics, purchase history, and transaction details. It is used to identify trends, patterns, and key drivers of customer behavior.
Tools andTechnologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
SQL (PostgreSQL / MySQL / SQL Server)
Power BI
Gamma (Report and Presentation)
Jupyter Notebook
Project Steps
Load and explore the dataset using Python
Perform Exploratory Data Analysis (EDA)
Clean and preprocess the data
Run SQL queries to answer business questions
Build an interactive Power BI dashboard
Create a detailed analytical report
Prepare a professional PPT using Gamma
Dashboard
The Power BI dashboard provides insights into customer segments, purchasing patterns, revenue trends, and key performance indicators with interactive filters.
Results and Insights
Identified high-value customer segments
Discovered key factors influencing purchasing behavior
Highlighted trends and anomalies in customer transactions
Provided actionable recommendations for business strategy
How to Run
Clone the repository
Install required Python libraries
Load the dataset
Run the Jupyter notebooks
Execute SQL scripts on your database
Open the Power BI file to view the dashboard.
